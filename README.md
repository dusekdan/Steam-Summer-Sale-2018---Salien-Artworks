# Steam-Summer-Sale-2018---Salien-Artworks

This repository contains a backup of those awesome Salien level backgrounds and Card Detail Artworks

I spent some time putting these together so I thought it would be a shame not to share them. Especially given the fact that they will probably become unavailable after the game ends.

**Extended Card Information**: You can download zip package with extended card information [directly from here](https://github.com/dusekdan/Steam-Summer-Sale-2018---Salien-Artworks/raw/master/Cards.zip).

**Salien game level backgrounds**: As it is not possible to store files over 100 MB in Github, to download background images, you have to click the green **Clone or download** button on the right above the file list and then click on **Download ZIP**. This whole repository (including `Cards` folder and `Cards.zip`) will be downloaded.
